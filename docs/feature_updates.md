# 🚀 Feature Updates (기능 업데이트 내용)

이 문서는 사용자 관점에서 추가되거나 변경된 주요 기능 업데이트 내역(Changelog)을 기록합니다.

---

## 📅 2024-12-31

### ✨ New Features (새로운 기능)
- **콜라주 레이아웃 (Collage Layouts)**
  - 4가지 새로운 레이아웃 지원: `1+2`, `2+1`, `1+3`, `2x2`.
  - 여러 상품 컷을 한 장의 매력적인 이미지로 생성 가능.
- **AI 프롬프트 추천 (AI Suggestion)**
  - 프롬프트 작성이 어려운 사용자를 위해 "✨ AI 추천" 버튼 추가.
  - 클릭 한 번으로 상품 정보에 기반한 고품질 프롬프트 자동 생성.
- **인트로 섹션 제어 (Intro Section Control)**
  - 상세페이지 상단의 상품 기본정보 영역 표시 여부를 간편하게 토글 스위치로 제어. (기존 좌측 사이드바에서 우측 상단으로 이동)

### 🎨 UI/UX Improvements (UI 개선)
- **섹션 추가 버튼 강화**: 더 크고 눈에 띄는 빨간색 버튼으로 디자인 변경.
- **사이드바 최적화**: 좌측 사이드바를 섹션 미니맵 전용으로 단순화하여 탐색 편의성 증대.
- **미니맵 디자인**: 각 섹션의 레이아웃 타입을 명확하게 보여주는 배지 디자인 적용.
- **콜라주 스타일 개선**: 이미지 간 경계선을 최소화(1-2px)하여 더욱 세련된 이미지 생성.

### 🛠️ Template Editor Enhancements (템플릿 편집기 개선)
- **자동 슬롯 생성**: 템플릿 수정 시 레이아웃을 변경하면(예: 2열 그리드), 필요한 이미지 슬롯(2개)이 자동으로 생성되도록 개선.
- **미리보기 편의성**: 템플릿 설정 화면의 '레이아웃 미리보기' 패널에서 섹션을 클릭하지 않고도 순서를 변경하거나 삭제할 수 있는 기능 추가.
